# Installation

## Package Manager

::: code-group

```bash [npm]
npm install vue-table-touch-scroll
```

```bash [pnpm]
pnpm add vue-table-touch-scroll
```

```bash [yarn]
yarn add vue-table-touch-scroll
```

:::

## CDN

```html
<script src="https://unpkg.com/vue-table-touch-scroll"></script>
```

## Requirements

- Vue 3.3.0 or higher
- Modern browser with ES2020 support
