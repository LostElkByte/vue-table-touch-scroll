# Getting Started

Vue Table Touch Scroll is a Vue 3 directive that enables touch and mouse drag scrolling for tables and other scrollable elements.

## Why?

Large tables on mobile devices or desktop can be difficult to navigate. This directive makes it easy to add intuitive drag-to-scroll functionality.

## Features

- 🚀 Simple directive-based API
- ⚡️ Lightweight with zero dependencies
- 📱 Touch and mouse support
- 🎨 Customizable options
- 🔧 Full TypeScript support
- 📦 Tree-shakable
