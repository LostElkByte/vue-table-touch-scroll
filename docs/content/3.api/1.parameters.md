# API 参考

## 指令

### vTableTouchScroll

Vue 3 自定义指令，为表格元素添加触摸滚动支持。

```ts
import { vTableTouchScroll } from 'vue-table-touch-scroll'
```

## 类型定义

### TableTouchScrollOptions

指令的配置选项接口。

```ts
interface TableTouchScrollOptions {
  /**
   * 是否启用自定义滚动
   * @default true
   */
  enabled?: boolean

  /**
   * 判定滚动方向前的位移阈值（像素）
   * 用于区分轻微抖动点击和有意识的拖拽
   * @default 5
   */
  threshold?: number

  /**
   * UI 库预设名称
   * 使用预设可以自动应用对应 UI 库的滚动容器选择器
   */
  preset?: TablePreset

  /**
   * 目标滚动元素的 CSS 选择器
   * 适用于自定义组件或需要精确控制的场景
   * 如果同时提供了 preset 和 selector，selector 优先级更高
   * 如果都未提供，则默认为绑定指令的元素本身
   */
  selector?: string
}
```

### TablePreset

支持的 UI 库预设类型。

```ts
type TablePreset =
  | 'element-plus'
  | 'ant-design-vue'
  | 'arco-design'
  | 'naive-ui'
  | 'primevue'
  | 'vuetify'
  | 'vxe-table'
```

## 常量

### UI_LIBRARY_SELECTORS

UI 库选择器预设映射表。

```ts
const UI_LIBRARY_SELECTORS: Record<TablePreset, string> = {
  'element-plus': '.el-scrollbar__wrap',
  'ant-design-vue': '.ant-table-body',
  'arco-design': '.arco-table-body',
  'naive-ui': '.n-scrollbar-container',
  'primevue': '.p-datatable-table-container',
  'vuetify': '.v-table__wrapper tbody',
  'vxe-table': '.vxe-table--body-inner-wrapper',
}
```

## 函数

### getSelectorByPreset

根据预设名称获取对应的选择器。

```ts
function getSelectorByPreset(
  preset: TablePreset
): string | undefined
```

**参数**

| 参数 | 类型 | 说明 |
|------|------|------|
| `preset` | `TablePreset` | UI 库预设名称 |

**返回值**

| 类型 | 说明 |
|------|------|
| `string \| undefined` | 对应的 CSS 选择器，如果预设不存在则返回 `undefined` |

**示例**

```ts
import { getSelectorByPreset } from 'vue-table-touch-scroll'

const selector = getSelectorByPreset('element-plus')
// 返回: '.el-scrollbar__wrap'
```

## 插件

### VueTableTouchScroll (默认导出)

Vue 插件，用于全局注册指令。

```ts
import VueTableTouchScroll from 'vue-table-touch-scroll'

app.use(VueTableTouchScroll)
// 注册后可在模板中使用 v-table-touch-scroll 指令
```

## 物理引擎参数

指令内部使用的物理引擎参数（不可配置）：

| 参数 | 值 | 说明 |
|------|-----|------|
| `DECAY` | `0.95` | 摩擦力/衰减率，值越低摩擦力越大 |
| `MIN_VELOCITY` | `0.05` | 停止动画的速度阈值（像素/毫秒） |
| `MAX_DT` | `32` | 最大帧间隔限制（毫秒），防止掉帧后产生大幅度位移跳变 |
| `DEFAULT_THRESHOLD` | `5` | 默认判定阈值（像素） |
| `INERTIA_STOP_DELAY` | `80` | 惯性保护延迟（毫秒），手指停顿超过此时间再松开，则不触发惯性 |
| `CLICK_BLOCK_DELAY` | `10` | 缓冲时间（毫秒），用于在触摸结束后捕获并拦截点击事件 |
| `SAFE_CLICK_VELOCITY` | `0.5` | 安全点击速度阈值（像素/毫秒），作用于用户点击正在滚动的列表进行"刹车"时 |

## 完整使用示例

### TypeScript + 全局注册

```ts [main.ts]
import { createApp } from 'vue'
import App from './App.vue'
import VueTableTouchScroll from 'vue-table-touch-scroll'

const app = createApp(App)
app.use(VueTableTouchScroll)
app.mount('#app')
```

### TypeScript + 局部使用

```vue [MyComponent.vue]
<script setup lang="ts">
import { vTableTouchScroll } from 'vue-table-touch-scroll'
import type { TableTouchScrollOptions } from 'vue-table-touch-scroll'

const options: TableTouchScrollOptions = {
  enabled: true,
  threshold: 5,
  preset: 'element-plus'
}
</script>

<template>
  <div v-table-touch-scroll="options">
    <el-table :data="tableData" />
  </div>
</template>
```

### 动态配置

```vue [DynamicComponent.vue]
<script setup lang="ts">
import { ref, computed } from 'vue'
import type { TableTouchScrollOptions } from 'vue-table-touch-scroll'

const isMobile = ref(true)
const customThreshold = ref(5)

const scrollOptions = computed<TableTouchScrollOptions>(() => ({
  enabled: isMobile.value,
  threshold: customThreshold.value,
  preset: 'element-plus'
}))
</script>

<template>
  <div v-table-touch-scroll="scrollOptions">
    <el-table :data="tableData" />
  </div>
</template>
```
