## 其他 UI 库

Vue Table Touch Scroll 支持所有桌面端 Vue UI 组件库中的 Table 组件。

已内置预设的组件库请参阅 [API 文档](/zh/api/parameters#tablepreset)，使用 `preset` 即可快速接入：

```vue
<template>
  <div v-table-touch-scroll="{ preset: 'arco-design' }">
    <a-table :columns="columns" :data="data" />
  </div>
</template>
```

## 没有预设？使用 Selector

对于没有内置预设的组件库，可以通过 `selector` 指定表格内部实际可滚动的容器元素。你需要通过浏览器开发者工具找到该表格组件中负责滚动的 DOM 元素的 CSS 选择器：

```vue
<template>
  <div v-table-touch-scroll="{ selector: '.your-table-scroll-container' }">
    <YourTable :data="data" />
  </div>
</template>
```
