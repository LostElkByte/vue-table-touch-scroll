# 使用指南

本页介绍 `v-table-touch-scroll` 指令的常见用法。完整的配置项和类型定义请参考 [API 文档](/zh/api/parameters)。

## 全局注册

```ts [main.ts]
import { createApp } from 'vue'
import App from './App.vue'
import VueTableTouchScroll from 'vue-table-touch-scroll'

const app = createApp(App)
app.use(VueTableTouchScroll)
app.mount('#app')
```

## 局部注册

如果不需要全局注册，可以在组件中直接导入指令：

```vue
<script setup lang="ts">
import { vTableTouchScroll } from 'vue-table-touch-scroll'
</script>

<template>
  <div v-table-touch-scroll="{ preset: 'element-plus' }">
    <el-table :data="tableData" />
  </div>
</template>
```

## 使用 UI 库预设

对于已内置预设的 UI 库，直接传入 `preset` 即可，无需手动查找滚动容器选择器：

```vue
<template>
  <!-- Element Plus -->
  <div v-table-touch-scroll="{ preset: 'element-plus' }">
    <el-table :data="tableData">
      <el-table-column prop="name" label="Name" />
    </el-table>
  </div>

  <!-- Ant Design Vue -->
  <div v-table-touch-scroll="{ preset: 'ant-design-vue' }">
    <a-table :data-source="dataSource" :columns="columns" />
  </div>

  <!-- VxeTable -->
  <div v-table-touch-scroll="{ preset: 'vxe-table' }">
    <vxe-table :data="tableData">
      <vxe-column field="name" title="Name" />
    </vxe-table>
  </div>
</template>
```

::callout{icon="lucide:info"}
支持的预设列表和对应选择器请查看 [API 文档 - TablePreset](/zh/api/parameters#tablepreset)。
::

## 使用自定义选择器

对于没有内置预设的 UI 库，通过 `selector` 指定表格内部的滚动容器：

```vue
<template>
  <div v-table-touch-scroll="{ selector: '.my-scroll-container' }">
    <MyTable />
  </div>
</template>
```

::callout{icon="lucide:triangle-alert" color="amber"}
`selector` 的优先级高于 `preset`。如果两者同时提供，将使用 `selector` 指定的元素。如果都未提供，则默认使用绑定指令的元素本身。
::

## 自定义物理参数

根据使用场景调整摩擦力和拖拽阈值：

```vue
<template>
  <!-- 更丝滑：高摩擦力（衰减慢），低阈值 -->
  <div v-table-touch-scroll="{ preset: 'element-plus', friction: 0.98, dragThreshold: 3 }">
    <el-table :data="tableData" />
  </div>

  <!-- 更精确：低摩擦力（衰减快），高阈值 -->
  <div v-table-touch-scroll="{ preset: 'element-plus', friction: 0.85, dragThreshold: 10 }">
    <el-table :data="tableData" />
  </div>
</template>
```

## 动态控制

传入 `false` 可完全禁用指令，适合根据设备类型动态切换：

```vue
<template>
  <div v-table-touch-scroll="isMobile ? { preset: 'element-plus' } : false">
    <el-table :data="tableData" />
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue'

const isMobile = ref(true)
</script>
```

也可以通过 `enabled` 属性控制：

```vue
<template>
  <div v-table-touch-scroll="{ enabled: isMobile, preset: 'element-plus' }">
    <el-table :data="tableData" />
  </div>
</template>
```

## 生命周期回调

```vue
<script setup lang="ts">
const scrollOptions = {
  preset: 'element-plus' as const,
  onScrollStart: () => {
    // 滚动开始：关闭浮层、记录开始时间等
  },
  onScrollEnd: () => {
    // 滚动结束：触发埋点、加载更多数据等
  },
}
</script>

<template>
  <div v-table-touch-scroll="scrollOptions">
    <el-table :data="tableData" />
  </div>
</template>
```
