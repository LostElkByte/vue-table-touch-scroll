# Vue Table Touch Scroll

一个专为移动端优化的 Vue 3 自定义指令，让你的表格在触摸设备上拥有原生般的流畅滚动体验。

## 核心特性

- 🚀 **高性能物理引擎** - 基于 Delta Time 的帧率归一化，确保在 60Hz/90Hz/120Hz 屏幕上有一致的惯性体感
- 📱 **智能方向锁定** - 防止手势过程中的对角线偏移，每次手势只锁定在一个轴向
- 🔄 **智能边缘检测** - 滚动到边界时自动将控制权交还给浏览器，支持 iOS 侧滑返回
- 🛑 **智能急停保护** - 区分"用户想点击"还是"用户想刹车"，仅在高速滑动时拦截点击事件
- 🎯 **零依赖** - 纯原生实现，无需额外依赖
- 🔧 **UI 库预设** - 内置 Element Plus、Ant Design Vue、Naive UI 等主流 UI 库的预设配置

## 适用场景

- Element Plus 的 `el-table` 组件
- Ant Design Vue 的 `a-table` 组件
- Naive UI 的 `n-data-table` 组件
- Arco Design、PrimeVue、Vuetify、VxeTable 等表格组件
- 任何需要优化移动端触摸滚动的自定义表格

## 快速开始

### 全局注册

```ts [main.ts]
import { createApp } from 'vue'
import App from './App.vue'
import VueTableTouchScroll from 'vue-table-touch-scroll'

const app = createApp(App)
app.use(VueTableTouchScroll)
app.mount('#app')
```

### 在模板中使用

```vue [MyComponent.vue]
<template>
  <div v-table-touch-scroll="{ uiLibrary: 'element-plus' }">
    <el-table :data="tableData">
      <el-table-column prop="name" label="Name" />
      <el-table-column prop="age" label="Age" />
      <el-table-column prop="address" label="Address" />
    </el-table>
  </div>
</template>
```

## 技术亮点

### 帧率归一化物理引擎

通过 Delta Time (dt) 计算，确保在不同刷新率屏幕上具有一致的惯性体感：

```ts
const rawDt = ctx.lastFrameTime ? now - ctx.lastFrameTime : 16.67
const dt = Math.min(rawDt, MAX_DT) // 限制最大帧间隔，防止掉帧后跳变
```

### 智能边缘检测

当滚动触达容器边界时，自动放弃接管，允许浏览器触发原生行为：

```ts
const atLeft = el.scrollLeft <= 1 && dx > 0
const atRight = el.scrollLeft >= maxScrollLeft - 1 && dx < 0
if (atLeft || atRight) isAtEdge = true
```

### 速度平滑处理

使用加权平均消除抖动，提供更自然的滚动体验：

```ts
ctx.velocity = ctx.velocity * 0.2 + instantV * 0.8
```
