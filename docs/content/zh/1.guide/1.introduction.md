## 功能特性

- 🚀 **Nuxt 3** - 直观的 Vue 框架
- 🎨 **Tailwind CSS** - 实用优先的 CSS 框架
- 📝 **MDC** - Markdown 组件，支持丰富内容
- 🌙 **深色模式** - 内置深色模式支持
- 🔍 **搜索** - 由 Nuxt Content 提供的全文搜索
- 📱 **响应式** - 移动端友好的设计

## 快速开始

### 安装

```bash
# 克隆模板
git clone <your-repo-url>

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

### 项目结构

```
.
├── components/      # Vue 组件
├── content/         # Markdown 文档
├── public/          # 静态资源
├── app.config.ts    # 应用配置
├── nuxt.config.ts   # Nuxt 配置
└── tailwind.config.js  # Tailwind 配置
```

## 编写文档

文档使用 Markdown 编写，支持 MDC（Markdown 组件）：

```md
::card
---
title: 我的卡片
---
卡片内容
::
```
