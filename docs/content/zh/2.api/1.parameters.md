# API 参考

## 指令

### vTableTouchScroll

Vue 3 自定义指令，为表格元素添加触摸滚动支持。

**全局注册：**

```ts
import VueTableTouchScroll from 'vue-table-touch-scroll'

app.use(VueTableTouchScroll)
// 注册后可在模板中使用 v-table-touch-scroll 指令
```

**局部导入：**

```ts
import { vTableTouchScroll } from 'vue-table-touch-scroll'
```

## 类型定义

### TableTouchScrollOptions

指令的配置选项接口。

| 属性 | 类型 | 默认值 | 说明 |
| ---- | ---- | ------ | ---- |
| `enabled` | `boolean` | `true` | 是否启用触摸滚动 |
| `preset` | `TablePreset` | - | UI 库预设名称，自动应用对应 UI 库的滚动容器选择器 |
| `selector` | `string` | - | 目标滚动元素的 CSS 选择器，优先级高于 `preset` |
| `friction` | `number` | `0.95` | 摩擦力/衰减率，值越大衰减越慢（滚动越丝滑），值越小即停感越强（建议范围：0.8 - 0.99） |
| `dragThreshold` | `number` | `5` | 判定滚动方向前的位移阈值 (px)，用于区分轻微抖动点击和有意识的拖拽 |
| `disableInertia` | `boolean` | `false` | 是否禁用惯性滚动，为 `true` 时手指松开即停止 |
| `clickBlockThreshold` | `number` | `0.5` | 急停点击拦截的速度阈值 (px/ms)，值越大越宽松，值越小越严格（建议范围：0.3 - 1.0） |
| `onScrollStart` | `() => void` | - | 滚动开始回调，在用户手指位移超过 `dragThreshold` 并锁定方向后触发 |
| `onScrollEnd` | `() => void` | - | 滚动结束回调，在惯性动画完全停止后触发 |

::callout{icon="lucide:info"}
滚动容器定位优先级：`selector` > `preset` > 指令绑定的元素本身。
::

**完整类型签名：**

```ts
interface TableTouchScrollOptions {
  enabled?: boolean
  preset?: TablePreset
  selector?: string
  friction?: number
  dragThreshold?: number
  disableInertia?: boolean
  clickBlockThreshold?: number
  onScrollStart?: () => void
  onScrollEnd?: () => void
}
```

### TablePreset

支持的 UI 库预设类型及对应的滚动容器选择器。

| 预设值 | 对应 UI 库 | 选择器 |
| ------ | --------- | ------ |
| `'element-plus'` | Element Plus | `.el-scrollbar__wrap` |
| `'ant-design-vue'` | Ant Design Vue | `.ant-table-body` |
| `'arco-design'` | Arco Design | `.arco-table-body` |
| `'naive-ui'` | Naive UI | `.n-scrollbar-container` |
| `'primevue'` | PrimeVue | `.p-datatable-table-container` |
| `'vuetify'` | Vuetify | `.v-table__wrapper tbody` |
| `'vxe-table'` | VxeTable | `.vxe-table--body-inner-wrapper` |

```ts
type TablePreset =
  | 'element-plus'
  | 'ant-design-vue'
  | 'arco-design'
  | 'naive-ui'
  | 'primevue'
  | 'vuetify'
  | 'vxe-table'
```

## 导出

### UI_LIBRARY_SELECTORS

UI 库选择器预设映射表常量。

```ts
import { UI_LIBRARY_SELECTORS } from 'vue-table-touch-scroll'

const selector = UI_LIBRARY_SELECTORS['element-plus']
// '.el-scrollbar__wrap'
```

### getSelectorByPreset

根据预设名称获取对应的 CSS 选择器。

```ts
import { getSelectorByPreset } from 'vue-table-touch-scroll'

const selector = getSelectorByPreset('element-plus')
// '.el-scrollbar__wrap'
```

**参数**

| 参数 | 类型 | 说明 |
| ---- | ---- | ---- |
| `preset` | `TablePreset` | UI 库预设名称 |

**返回值**

`string | undefined` — 对应的 CSS 选择器，如果预设不存在则返回 `undefined`。

## 内部物理引擎参数

指令内部使用的物理引擎常量（不可外部配置）：

| 常量 | 值 | 说明 |
| ---- | --- | ---- |
| `DEFAULT_FRICTION` | `0.95` | 默认摩擦力/衰减率 |
| `MIN_VELOCITY` | `0.05` | 停止惯性动画的速度阈值 (px/ms) |
| `MAX_DT` | `32` | 最大帧间隔限制 (ms)，防止掉帧后产生跳变 |
| `DEFAULT_DRAG_THRESHOLD` | `5` | 默认拖拽判定阈值 (px) |
| `INERTIA_STOP_DELAY` | `80` | 惯性保护延迟 (ms)，手指停顿超过此时间再松开则不触发惯性 |
| `SAFE_CLICK_VELOCITY` | `0.5` | 急停点击拦截的默认速度阈值 (px/ms) |
