## 如何创建演示组件

1. 在 `components/examples/` 中创建您的演示组件：

```vue
<!-- components/examples/my-demo.vue -->
<template>
  <div class="p-4">
    <h2>我的演示组件</h2>
    <button class="btn">点击我</button>
  </div>
</template>
```

2. 从 `components/examples/index.ts` 导出：

```ts
export { default as MyDemo } from './my-demo.vue'
```

3. 在文档中使用：

```md
::ComponentLoader{componentName="MyDemo" label="我的演示"}
::
```
