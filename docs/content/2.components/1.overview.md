## Built-in Components

### Layout Components

- `hero` - Hero section for landing pages
- `card` - Card component for content
- `card-group` - Group of cards
- `tabs` - Tabbed content

### Content Components

- `alert` - Alert/notification boxes
- `callout` - Callout blocks
- `steps` - Step-by-step guides
- `code-group` - Grouped code blocks
