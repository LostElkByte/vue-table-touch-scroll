## How to Create Demo Components

1. Create your demo component in `components/examples/`:

```vue
<!-- components/examples/my-demo.vue -->
<template>
  <div class="p-4">
    <h2>My Demo Component</h2>
    <button class="btn">Click me</button>
  </div>
</template>
```

2. Export it from `components/examples/index.ts`:

```ts
export { default as MyDemo } from './my-demo.vue'
```

3. Use it in your documentation:

```md
::ComponentLoader{componentName="MyDemo" label="My Demo"}
::
```
