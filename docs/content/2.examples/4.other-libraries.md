# 其他 UI 库示例

Vue Table Touch Scroll 支持多种 UI 库，以下是其他支持的库的使用示例。

## Arco Design

```vue
<template>
  <div v-table-touch-scroll="{ uiLibrary: 'arco-design' }">
    <a-table :columns="columns" :data="data" />
  </div>
</template>

<script setup lang="ts">
const columns = [
  {
    title: 'Name',
    dataIndex: 'name',
  },
  {
    title: 'Salary',
    dataIndex: 'salary',
  },
  {
    title: 'Address',
    dataIndex: 'address',
  },
  {
    title: 'Email',
    dataIndex: 'email',
  },
]

const data = [
  {
    key: '1',
    name: 'Jane Doe',
    salary: 23000,
    address: '32 Park Road, London',
    email: 'jane.doe@example.com',
  },
  {
    key: '2',
    name: 'Alisa Ross',
    salary: 25000,
    address: '35 Park Road, London',
    email: 'alisa.ross@example.com',
  },
]
</script>
```

## PrimeVue

```vue
<template>
  <div v-table-touch-scroll="{ uiLibrary: 'primevue' }">
    <DataTable :value="products" tableStyle="min-width: 50rem">
      <Column field="code" header="Code"></Column>
      <Column field="name" header="Name"></Column>
      <Column field="category" header="Category"></Column>
      <Column field="quantity" header="Quantity"></Column>
    </DataTable>
  </div>
</template>

<script setup lang="ts">
import { ref, onMounted } from 'vue'

const products = ref([])

onMounted(() => {
  products.value = [
    { code: 'P001', name: 'Product 1', category: 'Category A', quantity: 100 },
    { code: 'P002', name: 'Product 2', category: 'Category B', quantity: 200 },
  ]
})
</script>
```

## Vuetify

```vue
<template>
  <div v-table-touch-scroll="{ uiLibrary: 'vuetify' }">
    <v-table>
      <thead>
        <tr>
          <th class="text-left">Name</th>
          <th class="text-left">Calories</th>
        </tr>
      </thead>
      <tbody>
        <tr v-for="item in desserts" :key="item.name">
          <td>{{ item.name }}</td>
          <td>{{ item.calories }}</td>
        </tr>
      </tbody>
    </v-table>
  </div>
</template>

<script setup lang="ts">
const desserts = [
  { name: 'Frozen Yogurt', calories: 159 },
  { name: 'Ice cream sandwich', calories: 237 },
  { name: 'Eclair', calories: 262 },
]
</script>
```

## 自定义选择器

对于不支持的 UI 库或自定义组件，可以使用自定义选择器：

```vue
<template>
  <!-- 自定义表格组件 -->
  <div v-table-touch-scroll="{ selector: '.custom-table-body' }">
    <CustomTable>
      <div class="custom-table-wrapper">
        <div class="custom-table-header">
          <!-- 表头 -->
        </div>
        <div class="custom-table-body">
          <!-- 表格内容，这是实际滚动的容器 -->
          <div v-for="row in data" :key="row.id" class="table-row">
            <span>{{ row.name }}</span>
            <span>{{ row.value }}</span>
          </div>
        </div>
      </div>
    </CustomTable>
  </div>
</template>

<script setup lang="ts">
const data = [
  { id: 1, name: 'Item 1', value: 100 },
  { id: 2, name: 'Item 2', value: 200 },
  { id: 3, name: 'Item 3', value: 300 },
]
</script>

<style scoped>
.custom-table-body {
  overflow: auto;
  max-height: 400px;
}
</style>
```

## 动态选择器

根据条件动态选择滚动容器：

```vue
<template>
  <div v-table-touch-scroll="scrollOptions">
    <component :is="currentTableComponent" :data="tableData" />
  </div>
</template>

<script setup lang="ts">
import { ref, computed } from 'vue'
import type { TableTouchScrollOptions } from 'vue-table-touch-scroll'

const currentLibrary = ref('element-plus')

const scrollOptions = computed<TableTouchScrollOptions>(() => {
  // 根据当前使用的库返回不同的配置
  switch (currentLibrary.value) {
    case 'element-plus':
      return { uiLibrary: 'element-plus' }
    case 'ant-design-vue':
      return { uiLibrary: 'ant-design-vue' }
    case 'custom':
      return { selector: '.my-custom-scroll-container' }
    default:
      return { selector: '.default-scroll-container' }
  }
})

const tableData = ref([])
</script>
```
