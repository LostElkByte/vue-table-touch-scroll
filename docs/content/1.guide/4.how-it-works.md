## Core working principles

`v-table-touch-scroll` intercepts touch event flow and implements a complete in-directive scroll physics engine, covering gesture direction decision, velocity sampling, inertia decay, edge yielding, and device-specific strategies so desktop table components stay stable and follow-finger on mobile.

### Axis Locking

Automatically determines gesture intent when displacement exceeds the threshold, then locks the gesture to a single axis for its entire duration:

```ts
if (!ctx.gestureDirection) {
  if (Math.abs(dx) > threshold || Math.abs(dy) > threshold) {
    ctx.gestureDirection = Math.abs(dx) > Math.abs(dy) ? 'horizontal' : 'vertical'
    ctx.lockedDirection = ctx.gestureDirection
  }
}
```

### Touch History Queue

Uses sliding window sampling instead of traditional weighted averages, keeping only touch points from the last 50ms to calculate release velocity. This fundamentally eliminates sampling jitter on low-end devices and over-dense sampling on high refresh rate screens:

```ts
ctx.touchTracker.push({ time: now, x, y })
ctx.touchTracker = ctx.touchTracker.filter((p) => now - p.time <= 50)

const oldest = ctx.touchTracker[0]
const dt = Math.max(1, now - oldest.time)
if (ctx.gestureDirection === 'horizontal') {
  ctx.velocity = -(x - oldest.x) / dt
} else {
  ctx.velocity = -(y - oldest.y) / dt
}
```

### Frame Rate Normalization

Physics decay during inertia is normalized via Delta Time, ensuring the same swipe produces identical scroll distances regardless of screen refresh rate:

```ts
const rawDt = ctx.lastFrameTime ? now - ctx.lastFrameTime : 16.67
const dt = Math.min(rawDt, MAX_DT)
ctx.velocity *= ctx.friction ** (dt / 16.67)
```

### Synchronous Event Dispatching

Combined with dirty value detection, proactively dispatches `scroll` events to force immediate header synchronization, solving the visual tearing caused by asynchronous native `scroll` events:

```ts
if (el.scrollLeft !== prevX || el.scrollTop !== prevY) {
  el.dispatchEvent(new Event('scroll'))
}
```

### Edge Detection

Automatically yields control when scrolling reaches container boundaries, including detection for both horizontal and vertical axes and scenarios where no scroll is needed:

```ts
if (isHorizontal) {
  const maxScroll = el.scrollWidth - el.clientWidth
  const atLeft = el.scrollLeft <= 1 && dx > 0
  const atRight = el.scrollLeft >= maxScroll - 1 && dx < 0
  const noScroll = el.scrollWidth <= el.clientWidth
  if (atLeft || atRight || noScroll) isAtEdge = true
}
```

### Hybrid Device Lazy Hijacking

Uses a four-state machine to handle mixed environments — pure PC stays dormant, mobile activates immediately, and hybrid devices use sentinel events that only trigger full hijacking after confirming a real touch drag:

```ts
// Standby: only sentinel events, no style changes
// pending-active: touchstart recorded, but styles not hijacked yet
// active: overflow: hidden applied, full touch event binding
if (dx > ctx.dragThreshold || dy > ctx.dragThreshold) {
  activateHijacking(ctx)  // standby → active
}
```
