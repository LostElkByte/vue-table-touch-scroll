# Usage

This guide will walk you through the installation and basic usage of Vue Table Touch Scroll.

## Installation

Install the package using your preferred package manager:

:pm-install{name="vue-table-touch-scroll"}

## Basic Setup

### 1. Register the Directive

First, register the directive globally in your Vue application:

```ts [main.ts]
import { createApp } from 'vue'
import App from './App.vue'
import VueTableTouchScroll from 'vue-table-touch-scroll'

const app = createApp(App)
app.use(VueTableTouchScroll)
app.mount('#app')
```

### 2. Use the Directive

Add the `v-table-touch-scroll` directive to your table component:

```vue [MyComponent.vue]
<template>
  <el-table v-table-touch-scroll :data="tableData">
    <el-table-column prop="name" label="Name" />
    <el-table-column prop="age" label="Age" />
    <el-table-column prop="address" label="Address" />
  </el-table>
</template>

<script setup lang="ts">
import { ref } from 'vue'

const tableData = ref([
  { name: 'John', age: 25, address: 'New York' },
  { name: 'Jane', age: 30, address: 'London' },
  { name: 'Bob', age: 35, address: 'Paris' },
])
</script>
```

## Configuration Options

You can customize the behavior by passing options to the directive:

```vue
<template>
  <el-table 
    v-table-touch-scroll="{
      scrollSpeed: 1.5,
      enableMomentum: true,
      threshold: 10
    }"
    :data="tableData"
  >
    <!-- columns -->
  </el-table>
</template>
```

See the [API Parameters](/api/parameters) page for a complete list of available options.

## Local Registration

If you prefer to register the directive locally in a component:

```vue
<script setup lang="ts">
import { vTableTouchScroll } from 'vue-table-touch-scroll'

// Your component logic
</script>

<template>
  <el-table v-table-touch-scroll :data="tableData">
    <!-- columns -->
  </el-table>
</template>
```

## TypeScript Support

Vue Table Touch Scroll includes TypeScript definitions out of the box. No additional setup is required.

```ts
import type { TableTouchScrollOptions } from 'vue-table-touch-scroll'

const options: TableTouchScrollOptions = {
  scrollSpeed: 1.5,
  enableMomentum: true,
}
```
