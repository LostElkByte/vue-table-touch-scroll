# Vue Table Touch Scroll Documentation

Documentation for [Vue Table Touch Scroll](https://github.com/LostElkByte/vue-table-touch-scroll), a lightweight Vue 3 directive for touch scrolling in tables.

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install

# bun
bun install
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```

## Project Structure

This documentation is part of a monorepo project:

- `docs/` - Documentation site (Nuxt 3 + shadcn-docs-nuxt)
- `packages/` - Core packages and utilities
- `play/` - Development playground

## Features

- Interactive code examples with live preview
- Syntax highlighting with theme support
- Mobile-responsive design
- Dark/light mode toggle
- Component-based architecture
